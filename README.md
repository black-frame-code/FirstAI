# 🇺🇸 English

## 📖 About

Super AI Chatbot is a lightweight AI-inspired chatbot that runs entirely in the browser without any backend.

Instead of using cloud AI models, it uses:

- Pattern Learning
- Context Memory
- Similarity Search
- Character N-Grams
- Cosine Similarity
- LocalStorage Memory

Everything works locally inside the user's browser.

- 💬 Interactive chat interface
